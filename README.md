# hh-job-analyzer

программа собирает вакансии по запросу город - профессия/cпециальность с сайта hh ru
Делается запрос на каждый опыт(без опыта, опыт 1-3года, опыт 3-6 года, опыт болье 6 лет),
В идеальном варианте возращает 100 вакансий на каждый опыт по профессии по городу.

## Анализ.
Анализ осуществляется по данным собранным со всех вакансии по опыту.
На основе этих данных строятся графики
Так же выводится часто повторяющиеся слова в описаний вакансий


## как запускать:
--c - город
--v наименование специальности/профессии

```bash
python main.py --c Москва --v python-разработчик
```


если что-то не работает, автору похуй
